# Crypto Price Dashboard

Live cryptocurrency price dashboard with real-time data from the CoinGecko API, interactive charts, and auto-refresh.

## Features

- Live crypto prices with auto-refresh
- Interactive price charts (Recharts)
- Dark theme UI
- Responsive design with Tailwind CSS

## Tech Stack

- **Framework**: Next.js 14 (App Router)
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Data Fetching**: SWR
- **API**: CoinGecko (free tier)
- **Deployment**: Railway

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Deploy

Deployed automatically to Railway on push to `main`.
